## Build and Run Commands

```bash
# Build
dotnet build

# Build release
dotnet build --configuration Release

# Run
dotnet run --project CardAnalysis

# Publish self-contained executable
dotnet publish --configuration Release
```

No test projects are currently configured.

## Architecture

**CardAnalysis** is a .NET 10.0 Windows Forms desktop application (`WinExe`).

Entry point: `CardAnalysis/Program.cs` → initializes `Application` and runs `Form1`.

Current structure is a scaffold — `Form1.cs` is the main window with no business logic yet implemented. `Form1.Designer.cs` is auto-generated by the Visual Studio designer and should not be hand-edited.

The solution file is `CardAnalysis.slnx` (the newer XML-based solution format).
